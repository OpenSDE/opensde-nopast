sde-install(1)
==============
The OpenSDE Community <list@opensde.org>

NAME
----
sde-install - Takes care of installing software on you host system.

SYNOPSIS
--------
*sde install* 

DESCRIPTION
-----------
Install the *sde* wrapper on *$HOME/bin* which is expected to be
on your *$PATH*. The *sde* wrapper is only copied if the current
version is newer than an existing one.
