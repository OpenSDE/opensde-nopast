sde-help(1)
===========
The OpenSDE Community <list@opensde.org>

NAME
----
sde-help - provides help about the usage of the *sde* wrapper

SYNOPSIS
--------
*sde help* ['COMMAND']

DESCRIPTION
-----------
Provides help about the commands available for the *sde* wrapper
on the current tree. To see a full list of available commands, issue 
the *sde help* without any arguments. To get help on a specific command,
use that command as an argument
